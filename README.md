# Nebula-R
本模组是 Nebula on the ship 的分支版本 目前只有中文

## 停止更新说明
因为现在Nebula on the Ship处于转型阶段，源代码改动较大，加上Nebula-R的不稳定因素，因此决定将项目推倒重来一遍。

目前已知的bug已经修复不过来了，旧版的Nebula仍存在很多缺陷，并且Nebula on the Ship已经换上了新的架构体系，我会将制作的新内容全部转移到更新版本的Nebula上.

可能需要数月时间。我会尽量加紧制作速度，争取早日制作出稳定、新颖的Nebula分支。近期的停止更新也请谅解。
